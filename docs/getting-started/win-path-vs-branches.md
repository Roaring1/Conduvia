# Win-path vs branches

Conduvia has **one critical path** to the win, and a lot of optional life around
it. Knowing which is which keeps you from getting lost.

## Win-path

The **win-path** is the single chain of capabilities that ends with a rocket on
orbit. Every quest on it is marked <span class="qbadge qbadge-main">Win-path</span>,
and chapter-ending steps are marked <span class="qbadge qbadge-checkpoint">Checkpoint</span>.
If you only ever do these, you will still reach orbit.

```text
sticks + fiber -> cordage -> stone knife -> fire -> kiln + charcoal
  -> copper -> bronze -> bloomery -> wrought iron
  -> coke -> steam power -> blast furnace -> steel
  -> DC dynamo -> AC grid
  -> electrolysis (aluminium, chlor-alkali, ferroalloys, MG-silicon)
  -> polysilicon -> wafer -> chips / automation
  -> cryogenics (LOX) + titanium + superalloys
  -> rocket engine -> tanks + fuel -> stage -> vehicle -> ORBITAL LAUNCH
```

## Branches

Everything marked <span class="qbadge qbadge-side">Branch</span> is **optional**:
fishing, trapping, cooking, farming, decorative building, and the deeper
side-systems. Branches make your life easier or richer — better food, more
storage, comfort, automation conveniences — but the win doesn’t require them.

## Briefings & exchanges

- <span class="qbadge qbadge-info">Briefing</span> — a read-only intro or
  explainer. No crafting required.
- <span class="qbadge qbadge-exchange">Exchange</span> — the
  [token economy](../field-guide/rewards.md): trade category tokens for upgrades
  and bundles.

## How to use this

In a hurry or feeling lost? Follow only **Win-path** + **Checkpoint** quests down
[The Climb](../climb/index.md). Want to settle in and enjoy the world? Take the
branches. Both are legitimate ways to play — the guide just makes the line clear.
