# Controls

Conduvia is played on keyboard and mouse. The defaults:

| Key | Action |
|---|---|
| ++tab++ | Open / close your **Inventory** |
| ++j++ | Open / close the **Quest Log** |
| ++f++ | **Interact** — pick up items, use a station, open a container |
| ++r++ | **Recipes** — how to make the hovered item |
| ++u++ | **Uses** — what the hovered item is used *in* |
| ++a++ | **Favourite** the hovered item |
| ++k++ | **Bookmark** the hovered item |
| ++q++ | **Drop** one of the held item |
| ++g++ | **Sort** your inventory |
| ++m++ | Open the **Map** (fullscreen) |
| ++b++ | Drop a **map waypoint** at your position |
| ++n++ | Open the **waypoint list** |
| ++p++ | Open / close the **Menu** |
| ++v++ | **Push-to-talk** (voice chat) |
| ++shift++ | **Sprint** |
| ++ctrl++ | **Crouch** |
| ++c++ | **Slide** (while sprinting) |
| **Print Screen** | **Screenshot** |

## Two keys you’ll live in

- **++r++ (Recipes)** and **++u++ (Uses)** are your in-game version of this site.
  Hover anything, press ++r++ to see how it’s made, ++u++ to see what it feeds
  into. Lost? These two keys are the fastest way to orient.
- **++j++ (Quest Log)** always shows your current objectives and the next
  step on the [win-path](win-path-vs-branches.md).

!!! note "Note on ++r++"
    ++r++ shows the **Recipes** for whatever you're hovering. Sorting your
    inventory is a separate key, ++g++ — they don't conflict.

## Dying

Death isn’t the end of a run. You drop a **gravestone** where you fell and a
**waypoint** marks it on your map — go back to recover your things. Set up a
**bedroll** early: it claims your spawn point so you respawn nearby instead of
starting over far away. The [Survival chapter](../climb/0-survival.md) walks you
through both.
