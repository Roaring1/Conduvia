# Missing-chains roadmap — cleared

!!! success "This roadmap is complete"
    Every chain this page used to flag as “still owed” is now **built and in-world**.
    The list below is kept as a record of the build order that was followed; each
    entry now links to the field guide for the finished chain. There are no
    outstanding *new-chain* gaps — the current world ships **832 items · 685
    recipes · 153 stations** across all 27 chains.

This page was originally generated by `tools/missing_chains.py`, which walks the
recipe graph from mineable ore + world-gathered inputs and flagged every recipe
that couldn’t bottom out. All of those blocking inputs now have real chains.

## Cascade roots (resolved)

The big graph-blockers — `steel_plate_rolled`, `steel_beam`,
`metal_aluminum_ingot`, `alloy_aluminum_aerospace`, and the late-tier
electronics/space assemblies (`avionics_flight_computer`, `plc_*`,
`motor_ac_induction`) — all resolved once the metals + chemistry chains above
landed. They are reachable in the live graph today.

!!! info "Not real gaps"
    Wildlife carcasses, berries, acorns, hides, and tallow were only ever flagged
    as “NEEDS NEW CHAIN” because they are **world-gathered**, not crafted. They are
    not processing-chain gaps and never were.
