# The Climb

Eight chapters, gated in a fixed order. Each one unlocks a **real capability** — and the order is not arbitrary. You cannot skip a rung, because the physics of the next rung depends on the one below it. No silicon without industrial electricity; no electricity at scale without steel; no steel without coke-grade heat.

## The single critical win-path

Everything below is **one honest chain**. Everything *off* this line — farming, fishing, trapping, nuclear power, satellites — is optional. The guide marks <span class="qbadge qbadge-main">Win-path</span> vs <span class="qbadge qbadge-side">Branch</span> on every quest.

```text
sticks + plant fiber -> cordage -> stone knife -> fire -> kiln + charcoal
  -> copper smelt -> bronze (Cu+Sn) -> bloomery -> wrought iron
  -> coke (from coal) -> steam power -> blast furnace -> steel
  -> DC dynamo -> AC three-phase grid
  -> electrolysis: aluminum, chlor-alkali (Cl2/NaOH), ferroalloys, MG-silicon
  -> trichlorosilane -> polysilicon -> Czochralski wafer -> ICs / PLC automation
  -> cryogenic air separation (LOX); Kroll titanium; Inconel turbopump; Cu-Cr chamber
  -> rocket engine -> Al-Li tanks + RP-1 -> stage -> vehicle -> ORBITAL LAUNCH (win)
```
